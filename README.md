**ss-panel前后端一键安装脚本**  
`这是一个LAMP+sspanel+shadowsocks的一键安装脚本`
`只能在Centos6.X上使用`
`安装文件位于Vultr日本服务器上`  
`这是我第一次个项目 请大家多多包涵`

**使用方法**  
ssh下执行  wget mhans.cn/ssr.sh && bash ssr.sh  
将会提示输入授权码 授权码为990731  
如果不输入或输入错误将会无法安装  
输入成功会自动开始安装 根据提示进行填写即可  
等待5分钟左右环境将会安装完毕  
并打印出你的配置信息
